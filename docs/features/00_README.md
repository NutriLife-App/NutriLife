# NutriLife Documentation Pack

Цей архів містить оновлені документи для MVP застосунку NutriLife.

Оновлення в цій версії:
- основна тема: light-first
- повна підтримка dark mode
- основна мова: українська
- додаткова мова: англійська
- усі тексти мають проходити через localization layer
- всі feature docs узгоджені з цими рішеннями
